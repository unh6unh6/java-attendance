# java-attendance

출석 미션 저장소

### 추가적인 기능 요구사항

- 출석 수정은 과거 날짜의 출석 기록만 수정할 수 있다.

### 파일 입출력

- [ ] 파일 입출력으로 출석 시스템에 등록된 크루와 12월 출석 기록을 불러온다

### 출석 확인

- 기능
    - [x] 닉네임을 입력받는다
    - [x] 등교 시간을 입력받는다
    - [x] 출석을 한다
    - [x] 출석 종류를 계산한다
    - [x] 출석 기록을 출력한다
- 예외
    - [x] 존재하지 않는 크루일 경우 예외 메시지를 출력하고 프로그램을 종료한다
    - [x] 등교일이 아닐 경우 예외 메시지를 출력하고 프로그램을 종료한다
    - [x] 등교 시간이 HH:MM 형식이 아닐 경우 예외 메시지를 출력하고 프로그램을 종료한다
    - [x] 등교 시간이 아닐 경우 예외 메시지를 출력하고 프로그램을 종료한다
    - [x] 이미 출석한 경우, 수정 기능을 이용하도록 안내한다

### 출석 수정

- 기능
    - [x] 닉네임을 입력받는다
    - [x] 수정하려는 날짜(일)를 입력받는다
    - [x] 수정하려는 시간을 입력받는다
    - [x] 출석을 수정한다
    - [x] 수정된 출석 기록을 출력한다
- 예외
    - [x] 존재하지 않는 크루일 경우 예외 메시지를 출력하고 프로그램을 종료한다
    - [x] 등교일이 아닐 경우 예외 메시지를 출력하고 프로그램을 종료한다
    - [x] 등교일이 미래의 날짜라면 예외 메시지를 출력하고 프로그램을 종료한다
    - [x] 등교 시간이 HH:MM 형식이 아닐 경우 예외 메시지를 출력하고 프로그램을 종료한다
    - [x] 등교 시간이 아닐 경우 예외 메시지를 출력하고 프로그램을 종료한다

### 크루별 출석 기록 확인

- 기능
    - [x] 닉네임을 입력받는다
    - [x] 출석 기록을 확인한다
    - [x] 출석, 지각, 결석 횟수를 계산한다
    - [x] 경고, 면담, 제적 대상자인지 확인한다
    - [x] 크루별 출석 기록을 출력한다
- 예외
    - [x] 존재하지 않는 크루일 경우 예외 메시지를 출력하고 프로그램을 종료한다

### 제적 위험자 확인

- 기능
    - [ ] 전날까지의 크루 출석 기록을 바탕으로 제적 위험자를 파악한다
    - [ ] 제적 위험자는 제적 대상자, 면담 대상자, 경고 대상자 순으로 출력한다
        - [ ] 대상 항목별 정렬 순서는 지각을 결석으로 간주하여 내림차순한다
        - [ ] 출석 상태가 같으면 닉네임으로 오름차순 정렬한다
