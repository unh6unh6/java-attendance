# java-attendance

출석 미션 저장소

- [ ] 모든 입력은 입력받은 값이 공백이나 null이라면 예외를 발생시킨다

### 프로그램 시작

- [ ] 저장된 출석 기록을 불러온다
- [ ] 오늘 날짜를 안내하고 기능을 입력받는다
- [ ] 선택한 기능이 수행한다
    - [ ] 선택한 기능이 존재하지 않는다면 예외를 발생시킨다
- [ ] 사용자가 종료 입력을 할 때 까지 기능을 수행한 후 초기 화면으로 돌아간다

### 출석 확인

- [x] 오늘 날짜가 캠퍼스 운영일이 아니라면 예외를 발생시킨다
- [ ] 출석 확인하고자 하는 크루의 닉네임을 입력받는다
- [x] 크루의 닉네임으로 출석 기록을 가져온다
    - [x] 존재하지 않는 크루라면 예외를 발생시킨다
- [ ] 등교 시간을 입력받는다
    - [x] 시간 입력 포맷에 어긋난다면 예외를 발생시킨다
    - [x] 등교 시간이 캠퍼스 운영 시간이 아니라면 예외를 발생시킨다
- [x] 출석 기록을 추가한다
    - [x] 이미 출석한 경우 예외를 발생시키고 수정 기능을 이용하도록 안내한다
- [ ] 출석 결과를 출력한다

### 출석 수정

- [ ] 출석을 수정하고자 하는 크루의 닉네임을 입력받는다
- [x] 크루의 닉네임으로 출석 기록을 가져온다
    - [x] 존재하지 않는 크루라면 예외를 발생시킨다
- [ ] 출석을 수정하려는 날짜(일)을 입력받는다
    - [x] 날짜 입력 포맷에 어긋난다면 예외를 발생시킨다
- [x] 등교 시간을 입력받는다
    - [x] 시간 입력 포맷에 어긋난다면 예외를 발생시킨다
    - [x] 등교 시간이 캠퍼스 운영 시간이 아니라면 예외를 발생시킨다
- [ ] 출석 수정을 하려는 날짜가 오늘 또는 미래라면 예외를 발생시킨다
- [x] 출석 기록을 수정을 한다
    - [x] 출석 기록이 없다면 출석 기록을 추가한다
- [ ] 출석 수정 결과를 출력한다

### 크루별 출석 기록 확인

- [ ] 출석 기록을 확인하고자 하는 크루의 닉네임을 입력받는다
- [x] 크루의 닉네임으로 출석 기록을 가져온다
    - [x] 존재하지 않는 크루라면 예외를 발생시킨다
- [x] 출석 기록들을 조회한다
- [x] 출석 기록 및 상태들을 출력한다
- [x] 출석 상태별 횟수를 계산한다
- [x] 제적 위험자인지 확인한다
- [ ] 출석 상태별 횟수, 제적 위험 정보를 출력한다

### 제적 위험자 확인

- [ ] 전날까지의 크루 출석 기록을 바탕으로 제적 위험자를 파악한다
- [ ] 제적 위험자 크루를 정렬하여 출력한다
    - 제적 대상자, 면담 대상자, 경고 대상자 순서
    - 대상 항목별 정렬 순서는 지각을 결석으로 간주하여 내림차순
    - 출석 상태가 같으면 닉네임으로 오름차순
